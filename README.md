# Volto Contextmenu

The Volto Contextmenu Addon allows to use the ContextMenu Component for rendering a Contextmenu.

## Requirements

This addon need [`interaktiv.voltocontextmenu`](https://github.com/interaktivgmbh/interaktiv.voltocontextmenu) to be installed on the Plone backend.

## Installation

Add `@interaktiv.de/volto-contextmenu` to your package.json:

```json
"addons": [
  "@interaktiv.de/volto-contextmenu"
]

"dependencies": {
  "@interaktiv.de/volto-contextmenu": "*"
}
```

Download and install now by runnning

```sh
yarn install
```

Start Volto with:

```sh
yarn start
```
